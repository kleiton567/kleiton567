🎬 Novela Streaming

Plataforma completa para assistir novelas online com sistema de:

- catálogo
- temporadas
- episódios
- comentários
- favoritos
- painel admin

Inspirado no app Cine Novela.

- adicionar novelas

- adicionar episódios

- moderar comentários
